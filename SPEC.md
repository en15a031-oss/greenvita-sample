## 残りのAI感・修正課題

### 高優先度

- [ ] **商品画像プレースホルダー**
  - Hero: `.hero-product { content: "GV" }` → CSSテキストのみ
  - 実写風の商品イメージに差し替えるか、スタイリッシュなCSSビジュアルに改善
- [ ] **口コミ文の自然化（最重要）**
  - 現状: 5件中4件が★5、文章がきれいすぎる、全員が「続けています」で終わる
  - 目標: ★3〜4を混ぜる・飲みにくかった最初の感想・具体的な変化（「便通が良くなった」など）
- [ ] **Surveyのアンケート数値の説得力**
  - 「継続率92%」「体調87%」「紹介意向95%」が高すぎて不自然
  - 現実的な数値に調整（例：87%→72%程度）

### 中優先度

- [ ] **CTAセクションの購入フォーム**

- [ ] **免責・薬機法対応文言**
  - 「個人の感想であり〜」の表記は1行のみ → 健康食品LPとしてもう少し丁寧に
